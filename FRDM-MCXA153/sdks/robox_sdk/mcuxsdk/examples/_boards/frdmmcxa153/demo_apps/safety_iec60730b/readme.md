Hardware requirements
=====================
- Type-C USB cable
- FRDM-MCXA153 board
- Personal Computer

Overview
========
This application demonstrates use of certified NXP Safety Library 5.0 which meets the IEC60730 class B standard.

More information
================
More information about the Safety Library with the board specific Library/Example User's guide, release note, certificate and other documentation: http://www.nxp.com/iec60730
