## Overview
The power demo project demonstrates the capabilities of the power driver, to go into multiple power states (e.g. sleep mode, deep sleep mode), 
to wake up the chip from different sources (e.g. RTC timer, OS timer, Wake up button), and to calibrate device FROs (e.g. 32KHZ, 1MHZ, 12MHZ)
