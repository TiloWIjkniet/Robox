# dcif_clut_4bpp

## Overview
The clut_4bpp project is designed to demonstrate the Color Look-Up Table (CLUT) feature using the 4BPP format on the background layer. 
If the program runs successfully, a 400x400 square will sequentially display 16 different colors.

## Supported Boards
- [IMX943-EVK](../../../_boards/imx943evk/driver_examples/dcif/dcif_examples_readme.md)
