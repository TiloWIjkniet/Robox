# rtc_freerun

## Overview
The RTC project is a demonstration program of the SDK RTC driver. It sets RTC work on free run mode. It enables both alarm0 and alarm1. Alarm0 works in repeated mode, alarm occurs every 2 seconds. Alarm1 works in single shot mode, alarm occurs in 5th second. Message ahout which alarm occurs will be logged out.
