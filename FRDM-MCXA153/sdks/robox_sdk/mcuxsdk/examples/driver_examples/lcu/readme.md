# LCU

## Overview

The LCU Example project is to demonstrate usage of the KSDK lcu driver.
In this example, The LCU will configure the LUT table to LCU OUTPUT 
and trigger interrupt when a LUT event is generated.

## Running the demo

When the example runs successfully, you can see the similar information from the terminal as below.

LCU interrupt Example started !

LCU interrupt is occurred

LCU interrupt flag is cleared !

LCU interrupt example is finished !
