# emios_simple_pwm

## Overview

The eMIOS project is a simple demonstration example of the SDK eMIOS driver to use eMIOS as PWM generator.  
It configures eMIOS UC (Unified channel) as Output PWM Buffered (OPWMB) mode and generate one 10kHz, 25% duty cycle PWM.  

## Supported Boards
- [FRDM-MCXE31B](../../../_boards/frdmmcxe31b/driver_examples/emios/simple_pwm/example_board_readme.md)
