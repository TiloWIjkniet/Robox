# emios_input_capture

## Overview

The eMIOS project is a simple demonstration example of the SDK eMIOS driver to use eMIOS as pulse width measurement.  
Example will print pulse width in ticks after a successful capture.

## Supported Boards
- [FRDM-MCXE31B](../../../_boards/frdmmcxe31b/driver_examples/emios/input_capture/example_board_readme.md)
