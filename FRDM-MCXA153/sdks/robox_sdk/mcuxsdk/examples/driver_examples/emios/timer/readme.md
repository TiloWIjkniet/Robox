# emios_timer

## Overview

The eMIOS project is a simple demonstration example of the SDK eMIOS driver to use eMIOS as a timer.  
It sets up the eMIOS Modulus Counter to trigger an interrupt every 1 millisecond, and print '1s elapsed' every 1 second.
