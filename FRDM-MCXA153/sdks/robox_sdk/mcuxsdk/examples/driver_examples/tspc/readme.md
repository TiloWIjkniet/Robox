# TSPC

## Overview

The TSPC Example project is to demonstrate usage of the KSDK tspc driver.
In this example,  TSPC will put the two pads into one group, then disable the output
buffer of one pad, and the output buffer of the other pad will change accordingly.

## Running the demo

When the example runs successfully, you can see the similar information from the terminal as below.

TSPC Example started !

PAD2 interrupt is occurred !

PAD2 interrupt flag is cleared !

TSPC example is finished !
