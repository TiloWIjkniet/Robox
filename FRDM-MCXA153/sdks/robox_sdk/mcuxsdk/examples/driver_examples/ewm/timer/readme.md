# ewm_timer

## Overview
The EWM TIMER Example project is to demonstrate usage of the KSDK EWM driver.
In the example, EWM counter is continuously refreshed until timer is timeout.
Once the timer is timeut, EWM counter will expire and interrupt will be generated.

## Supported Boards
- [IMX943-EVK](../../_boards/imx943evk/driver_examples/ewm/example_board_readme.md)
