# siul2_input_dma

## Overview
The siul2 Example project is a demonstration program that uses the KSDK software to manipulate the external DMA request.
The example is supported by trigger DMA request using the external REQ pins. The example uses the software button to 
trigger a DMA transfer.

## Supported Boards
- [FRDM-MCXE31B](../../../_boards/frdmmcxe31b/driver_examples/siul2/input_dma/example_board_readme.md)
