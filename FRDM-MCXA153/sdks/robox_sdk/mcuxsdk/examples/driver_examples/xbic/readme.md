# XBIC

## Overview

The XBIC Example project is to demonstrate usage of the KSDK xbic driver.
In this example, The XBIC will enable master port and slave port error detection.
Then inject errors into the communication and check whether the error information is correct.

## Running the demo

When the example runs successfully, you can see the similar information from the terminal as below.

XBIC Example started !

XBIC error Injected successfully !

XBIC error data received successfully !

XBIC Example finished !
