.. _examples:

Examples
########


The following documents provide an overview of the examples included in the MCUXpresso SDK, located under the ``mcuxsdk/examples`` directory.

Each example typically includes:

- A brief overview of its purpose and functionality
- Hardware and software requirements (if any)
- Step-by-step instructions for running the demo on supported hardware
- Known issues, if applicable
- A list of supported boards

If a supported board includes a link, click it to access the board-specific document. These documents outline board-specific setup requirements, including jumper settings, cable connections, and software configurations necessary for correct operation.
