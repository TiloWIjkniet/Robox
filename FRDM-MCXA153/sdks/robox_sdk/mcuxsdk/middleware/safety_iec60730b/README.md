## NXP IEC 60730 Class B Safety Library

Safety is an obligatory part of many applications. It is controlled by international standards related to electrical devices. The affected devices are those that, in case of failure, can cause injury, death, pollution, or other dangerous situations.

### Documentation

The documentation is provided on [IEC 60730 Safety Standard for Household Appliances](https://www.nxp.com/applications/technologies/functional-safety/iec-60730-safety-standard-for-household-appliances:APIEC60730) web.
