# MCUXpresso SDK : CMSIS

## Overview
This repository is for MCUXpresso SDK CMSIS integration. It contains the components CMSIS 6, CMSIS-DSP, CMSIS-NN and CMSIS-RTX projects. This repository is a part of the MCUXpresso SDK overall delivery which is composed of several sub-repositories/projects. Navigate to the top/parent repository for the complete delivery of MCUXpresso SDK.

## Documentation
Overall SDK Documentation can be reviewed here: [MCUXpresso SDK Online Documentation](https://mcuxpresso.nxp.com/mcuxsdk/latest/html/introduction/README.html)

Visit [CMSIS - Documentation](https://mcuxpresso.nxp.com/mcuxsdk/latest/html/develop/architect.html#cmsis-support) to review the contents about this sub-repo.
